# Diagnóstico TRL / IRL

Prototipo de una plataforma web para evaluar el nivel de madurez tecnológica (TRL/IRL) de emprendimientos científicos y traducirlo en acciones concretas de modelo de negocio.

## Objetivo

Validar la hipótesis:

> Si se ofrece un diagnóstico TRL/IRL con traducción a acciones de negocio, entonces al menos el 20% de los usuarios hará clic en “ver resultados completos”.

## Metodología

Se implementó un experimento tipo:
- Mago de Oz (resultados simulados)
- Fake Door (medición de interés mediante clic)

## Funcionamiento

1. Usuario completa cuestionario
2. Se muestra resultado preliminar
3. Se invita a ver resultados completos
4. Se mide el número de clics

## Despliegue

Este proyecto puede desplegarse con GitHub Pages.

## Créditos

El diseño del prototipo fue desarrollado por Andrea Gregorio como parte de un proceso de validación de emprendimiento de base científica para la clase de Temas Selectos de Filosofía de la Física: Ciencia e Industria.

La generación de código fue asistida mediante el uso de ChatGPT (OpenAI), utilizado como herramienta de prototipado rápido.

## Nota

Este es un prototipo experimental y no representa un sistema automatizado real de evaluación TRL/IRL.
